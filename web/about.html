About

What is Epic Free Games?

Epic Free Games is a simple tracker for free games on the Epic Games Store. Epic releases new free games regularly, and this site helps you find them quickly and clearly.

Key features

Current free games

See the games you can claim right now.

Upcoming free games

Preview the next free games so you do not miss them.

Game details

View images, genres, and free period information.

Direct links

Jump to the Epic Store page for each game.

Dark mode

Switch between light and dark themes, saved to your browser.

Responsive design

Optimized for desktop, tablet, and mobile.

Update schedule

Epic Games Store typically updates free games every Thursday (KST). This site refreshes data daily to stay current.

  • Free games data: updated daily
  • Images and metadata: synced automatically
  • Free periods: based on Epic Games Store schedules

Data sources

All game information shown here is based on public data from the Epic Games Store. Game images, titles, genres, and free periods are collected via the Epic Games Store API.

All game trademarks and intellectual property belong to their respective owners. This site is for informational purposes and is not affiliated with Epic Games.

How to use

Epic Free Games is available without registration or login. To claim a game, you will need an Epic Games Store account. Use the "View on Epic Store" button on a game card to open the store page.
